0.з®Җд»Ӣ

еңЁз»ҳеҲ¶дёӨдёӘжҲ–еӨҡдёӘеҮҪж•°ж—¶пјҢдҪ еҸҜиғҪеёҢжңӣжҳҫејҸең°и®ҫзҪ®жҜҸдёӘз»ҳеӣҫиЎҢзҡ„ж ·ејҸгҖӮ
дҪ иҝҳеҸҜд»ҘдҪҝз”Ёеқҗж ҮиҪҙжқҘйҡҗи—ҸдёҖдёӘжҲ–дёӨдёӘиҪҙпјҢе°ұеғҸжҲ‘еңЁз¬¬дәҢдёӘе’Ңз¬¬еӣӣдёӘеӣҫдёӯжүҖеҒҡзҡ„йӮЈж ·гҖӮ

еҸҜд»ҘдҪҝз”ЁGridз»ҷеӣҫеғҸеҠ зҪ‘ж јпјҢзҪ‘ж јзҡ„ж ·ејҸеҸҜд»Ҙз”ұеҮҪж•°иҮӘеҠЁз”ҹжҲҗпјҢд№ҹеҸҜиҮӘе®ҡд№үгҖӮ

еӣҫеғҸпјҡ

еӣҫеғҸзҡ„иҫ№жЎҶ Frame ,FrameStyle, FrameLabe

FrameгҖҒFrameStyleе’ҢFrameLabelе…Ғи®ёдҪ дҪҝз”Ёиҫ№жЎҶе’Ңж ҮзӯҫеҜ№еӣҫеҪўиҝӣиЎҢжіЁйҮҠгҖӮ

GraphicsGrid[
GraphicsRow[{
Plot[Exp[Sin@x], {x, 0, 2 Pi}, Frame -> True, FrameLabel -> "\!\(\*SuperscriptBox[\(e\), \(sin\\\ x\)]\)"],
Plot[Exp[Cos@x], {x, 0, 2 Pi}, Frame -> True, 
FrameLabel -> "\!\(\*SuperscriptBox[\(e\), \(cosx\)]\)", 
FrameStyle -> Directive[Gray, Thick]]
}]

еӣҫеғҸпјҡ

еӣҫеғҸзӘҒеҮәзӮ№Mesh

Meshе…Ғи®ёдҪ еңЁеӣҫдёӯй«ҳдә®жҳҫзӨәзү№е®ҡзҡ„зӮ№гҖӮ
Mesh->FullжүҖжңүе°ҶзӘҒеҮәзӮ№йҮҮж ·иҖҢз»ҳеҲ¶еӣҫеҪў.
Mesh->nе°ҶдҪҝз”Ёn дёӘеҗҢж ·и·қзҰ»зҡ„еҲҶйҡ”зәҝ,жқҘзӘҒеҮәеӣҫеғҸ.
Mesh->Automaticе°ҶдҪҝз”Ёзӯүи·қзҡ„зӮ№гҖӮ

GraphicsGrid[
Partition[Table[
Plot[0.5 Sin[2 x], {x, 0, 2 Pi}, Mesh -> m, Frame -> True, 
ImageSize -> Small,
PlotLabel -> "Mesh->" <> ToString@m],
{m, {None, Automatic, All, Full, 16, 50}}], 2], Spacings -> 0]

еӣҫеғҸпјҡ

еӣҫеғҸзҡ„иҢғеӣҙPlotRange

PlotRangeжҳҜдёҖдёӘйҮҚиҰҒзҡ„йҖүйЎ№пјҢе®ғеҸҜд»ҘжҺ§еҲ¶еӣҫдёӯеҢ…еҗ«е“Әдәӣеқҗж ҮгҖӮ
Automaticи®©MathematicaеҶіе®ҡжңҖеҘҪзҡ„йҖүжӢ©пјҢжүҖжңүжҢҮе®ҡжүҖжңүзӮ№е®һйҷ…з»ҳеҲ¶.
FullжҢҮе®ҡж•ҙдёӘиҢғеӣҙгҖӮ

GraphicsGrid[Partition[
Table[
Plot[Sqrt[100. - x^2], {x, 0, 100},
PlotRange -> r, Frame -> True,
FrameLabel -> "з»ҳеӣҫиҢғеӣҙ" <> ToString@r],
{r, {Automatic, All, Full, {{0, 20}, {0, 20}}}}
], 2], Spacings -> 0]

еӣҫеғҸпјҡ

еӣҫеғҸе®Ҫй«ҳжҜ”AspectRatio

AspectRatioжҺ§еҲ¶зқҖең°еқ—зҡ„е®Ҫй«ҳжҜ”
й»ҳи®ӨеҖјдёәGoldenRatioпјҲй»„йҮ‘жҜ”дҫӢпјүпјҢ
Automaticзҡ„еҖјдҪҝз”Ёеқҗж ҮеҖјжқҘзЎ®е®ҡй•ҝе®ҪжҜ”гҖӮ

GraphicsGrid@Partition[
Table[
Plot[Sqrt[100. - x^2], {x, 0, 10}, AspectRatio -> a, Frame -> True,
FrameLabel -> "е®Ҫй«ҳжҜ”жҳҜ" <> ToString[TraditionalForm@a]],
{a, {GoldenRatio, Automatic, 1.2, 0.4}}], 2]

еӣҫеғҸпјҡ

еӣҫеғҸзҡ„еЎ«е……Filling

жңүж—¶еҖҷдҪ жғіеҺ»ејәи°ғдҪ зҡ„еӣҫеғҸзҡ„жҹҗдёӘйғЁдҪҚпјҢжҲ–иҖ…иҜҙдёӨжқЎжӣІзәҝеӣҙеңЁдёҖиө·зҡ„йқўз§ҜгҖӮ
жӯӨж—¶дҪ з”ЁFillingеҸҜд»ҘеҒҡеҲ°иҝҷдёҖзӮ№
дҪ еҸҜд»Ҙе°Ҷе…¶и®ҫзҪ®дёәвҖңTop(йЎ¶йғЁ)вҖқд»Ҙд»ҺжӣІзәҝеҗ‘дёҠеЎ«е……,вҖңBottom(еә•йғЁ)вҖқд»Ҙд»ҺжӣІзәҝеҗ‘дёӢеЎ«е……пјҢвҖңAxis(иҪҙ)вҖқд»Ҙд»ҺиҪҙеҗ‘жӣІзәҝеЎ«е……пјҢжҲ–и®ҫзҪ®дёәж•°еҖјд»Ҙд»ҺжӣІзәҝеҗ‘д»»ж„Ҹyж–№еҗ‘зҡ„еҖјеЎ«е……гҖӮ

GraphicsGrid@Partition[
 Table[Plot[Sin@x, {x, 0, 2 Pi}, Filling -> f, Frame -> True,
  FrameLabel -> "еЎ«е……жҳҜ" <> ToString@f], {f, {Top, Bottom, Axis, 
 0.5}}], 2]

еӣҫеғҸпјҡ

FillingStyleе…Ғи®ёдҪ жҺ§еҲ¶еЎ«е……зҡ„йўңиүІе’ҢдёҚйҖҸжҳҺеәҰ
еҪ“еӨҡдёӘеҮҪж•°зҡ„еҢәеҹҹйҮҚеҸ ж—¶пјҢжҢҮе®ҡдёҚйҖҸжҳҺеәҰжҳҜеҫҲжңүз”Ёзҡ„гҖӮ

Plot[{Cosh[x], Cosh[3 x]}, {x, -1, 1}, Filling -> Top,
